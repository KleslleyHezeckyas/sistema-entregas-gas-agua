# Gerenciador de Tarefas em Go

Este projeto é um **Gerenciador de Tarefas** desenvolvido em **Golang**, executado diretamente no terminal.  
O sistema permite **adicionar**, **visualizar** e **remover** tarefas de forma simples e funcional.

Embora eu ainda esteja aprendendo Go, este projeto foi uma excelente oportunidade para explorar a linguagem, compreender sua estrutura e praticar conceitos fundamentais de programação.

---

## Funcionalidades

- Adicionar novas tarefas  
- Listar todas as tarefas  
- Remover tarefas existentes  
- Executar o sistema em loop até que o usuário escolha sair  

---

## Conceitos praticados

Durante o desenvolvimento, foi possível reforçar e compreender melhor:

- Estruturas de repetição e condicionais  
- Manipulação de slices e entrada de dados do usuário  
- Organização lógica de um programa em Go  
- Boas práticas na escrita de código limpo e legível
